Stack For Yourself

You, developer. For free. On your own.

This site aggregates resources for web development from around the web. Browse and bookmark :) if you have cool resources to contribute, add your resources by contributing your links via comments on the appropriate pages!

Learning to code is really damn hard, so remember that it's not just you, and don't give up. Your first 100 hours are probably going to be especially difficult, if you're truly challenging yourself, but you'll do better if you always know how the specific thing you're learning fits into your overall plan.

If you're absolutely a raw beginner, start with Paul Ford's "What is Code?" essay. And be sure to understand how to best use this site by checking out the roadmap!
